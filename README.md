# Калькулятор матриц, СЛАУ и векторов (Go)

Лёгкий веб-калькулятор без тяжёлых GUI-библиотек (только стандартная библиотека Go).

## Возможности

- Матрицы: сложение, вычитание, умножение на число, перемножение, транспонирование, определитель, ранг.
- СЛАУ: метод Гаусса, метод Крамера, матричный метод (через обратную матрицу).
- Векторы: сложение, вычитание, умножение на число, скалярное, векторное и смешанное произведения.
- Показ промежуточных действий и результата.
- Сообщения об ошибках при математической невозможности операции или сбое вычисления.

## Запуск

```bash
go run ./cmd/calcmatxgo
```

Откройте: <http://localhost:8080>

## Ввод

- Числа в строке разделяйте пробелами.
- Матрицу вводите построчно: каждая строка на новой строке.
- Векторы вводите в одну строку (или в textarea) через пробел.
